# Simple Inferno Isomorphic (SSR) Webpack template

Made with:

- Inferno
- Inferno Router
- Express
- TypeScript
- Webpack

# Installation

## Local

```
yarn
yarn start
```

and goto http://localhost:1234

## Docker

```
sudo docker build . --tag iit
sudo docker run -it --publish 1234:1234 iit
```

# Develop

```
yarn
yarn dev # Any save to the tsx or css files will re-compile.
```
